== Description ==

[Plaay](http://www.plaay.com.br) é um plugin para adicionar músicas, playlists e conteúdos musicais de artistas nos seus posts de um jeito fácil e rápido.

[youtube http://www.youtube.com/watch?v=OlxPgmyfbj4]

Para mais informações, visite [Plaay](http://www.plaay.com.br) 


Vantagens:

* Todo conteúdo musical do YouTube há um clique e de um jeito muito mais clean

* Deixa seu conteúdo mais atrativo, fazendo que visitantes fiquem até 40% mais tempo na sua página

* Adicionar playlists, músicas e artistas nas postagens diretos do painel do *editor de posts*

* Widget interativo e leve

* Interação social com os leitores do site através do Plaay - eles podem te seguir e acompanhar o que você escuta


== Installation ==

1. Instale o Plaay através do diretório de plugins do WordPress.org, ou fazendo o upload dos arquivos para o seu servidor

2. Ative o plugin no seu WordPress

3. Pronto! Agora é só usar! 

[youtube http://www.youtube.com/watch?v=OlxPgmyfbj4]


== Frequently Asked Questions ==

= Eu preciso ter uma conta no Plaay para utilizar o plugin? =

Não, você não precise ter uma conta no [Plaay](http://plaay.com.br) para utilizar o plugin, porém se quiser incluir uma playlist sua dentro de um post, você precisa cria-la através do site para então poder seleciona-la no plugin. Você pode incluir músicas, albuns e playlists de outros usuários sem ter sua própria conta.
